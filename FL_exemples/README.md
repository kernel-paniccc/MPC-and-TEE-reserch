# FL Examples

Small experiments with `CrypTen`, `PyTorch`, and basic distributed communication.

## Files
- `crypten_simple.py` - minimal example of encrypting a tensor and decrypting the result.
- `crypten_mul.py` - multiplication of two private tensors in a two-process MPC setup using `CrypTen`.
- `torch_broadcast.py` - simple `broadcast` example with `torch.distributed`.
