# MPC-and-TEE-research

Repository for RnD in MPC and TEE area with examples and scratch scripts.

## Contents
- `MP-SPDZ/` — upstream MP-SPDZ codebase.
- `exemples_spdz/` — simple `.mpc` examples.
- `FL_exemples/` — federated learning examples.
- `scripts/` — helper scripts.

## Quick start
- Check the MPC examples in `exemples_spdz/` and the sources under `MP-SPDZ/`.
- Federated learning examples are under `FL_exemples/`.
